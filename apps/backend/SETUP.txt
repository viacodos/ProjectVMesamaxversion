BACKEND SETUP INSTRUCTIONS
==========================

1. CREATE .env FILE
   - Copy .env.example to .env
   - Update with your actual credentials:
     * DB_PASSWORD=Aabid2004@
     * JWT_SECRET=generate_a_random_secret_key_here
     * FRONTEND_URL=http://localhost:8080

2. INSTALL DEPENDENCIES (Already done)
   npm install

3. TEST THE NEW BACKEND
   node server-new.js

4. ONCE TESTED, REPLACE OLD SERVER
   - Backup: copy server.js to server-old.js
   - Rename: server-new.js to server.js
   - Update package.json start script if needed

5. API ENDPOINTS AVAILABLE:

   PUBLIC ENDPOINTS:
   - GET  /api/health
   - GET  /api/destinations
   - GET  /api/packages
   - GET  /api/packages/:id
   - GET  /api/activities

   - GET  /api/itinerary/:id/map-data
   - GET  /api/map/destinations
   - GET  /api/map/hotels
   - GET  /api/map/package/:id/route

   ADMIN ENDPOINTS (Require JWT):
   - POST   /api/admin/destinations
   - PUT    /api/admin/destinations/:id
   - DELETE /api/admin/destinations/:id
   - POST   /api/admin/hotels
   - GET    /api/admin/destinations
   - GET    /api/admin/hotels
